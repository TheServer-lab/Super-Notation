üéì Getting Started

For Authors (Creating Docs)

  1. Open sn-editor.html in your browser
  2. Click "üìã Templates" and choose a template type
  3. Edit content using the toolbar or type commands directly
  4. Watch the live preview update in real-time
  5. Click "üíæ Save" to download your .sn file
  6. View in sn-browser.html to see final result

For Readers (Viewing Docs)

  1. Open sn-browser.html in your browser
  2. Click "Choose File" or drag & drop a .sn file
  3. Or paste a URL to load from the web
  4. Read the beautifully formatted document
  5. Use section links and "Next" for navigation

For Publishers (Hosting Online)

  1. Create your .sn documentation files
  2. Upload sn-browser.html to your web server
  3. Upload your .sn files to the same location
  4. Share the browser URL with users
  5. Users can view all docs in their browser
